Nível Básico — 15 min

Crie CorridaBasica.java. Declare variáveis fixas de uma corrida de Fórmula e calcule estatísticas simples usando apenas operadores aritméticos.

Dados fixos da corrida:
  • int totalVoltas = 57
  • int voltasCompletas = 38
  • double tempoVoltaMedia = 91.4 — segundos
  • double combustivelInicial = 110.0 — litros
  • double consumoPorVolta = 1.85 — litros/volta

Calcule e exiba:
  1. Voltas restantes: totalVoltas - voltasCompletas
  2. % corrida concluída: voltasCompletas * 100 / totalVoltas (inteiro) e como double com cast
  3. Combustível consumido: voltasCompletas * consumoPorVolta
  4. Combustível restante: combustivelInicial - consumido
  5. Tempo já decorrido: voltasCompletas * tempoVoltaMedia → converta para min:seg

Saída esperada:
══ STATUS DA CORRIDA ════════ Voltas : 38 / 57 Restantes : 19 voltas Progresso : 66% (66.67%) Combustível: 29.70 L usados / 80.30 L restantes Tempo : 57 min 49 seg decorridos
Nível Médio — 20 min

Crie PitStop.java. Mantenha os dados fixos e adicione a lógica de pit stop usando operadores relacionais, lógicos e módulo.

Regras do pit stop (calcule com booleanos):
  • boolean precisaCombustivel = combustivelRestante < 30.0
  • boolean precisaPneu = voltasCompletas % 15 == 0 — troca a cada 15 voltas
  • boolean dentroDoSafetyCar = voltasCompletas >= 20 && voltasCompletas <= 25
  • boolean pitObrigatorio = precisaCombustivel || precisaPneu
  • boolean pitIdeal = pitObrigatorio && !dentroDoSafetyCar

Use incremento/decremento:
  • Simule 3 pit stops: a cada pit, paradas++, voltasCompletas += 1
  • Mostre o estado antes e depois de cada parada

Teste com voltasCompletas=30:
→ precisaCombustivel=false, precisaPneu=true (30%15==0), pitObrigatorio=true
Nível Avançado — 25 min

Crie CorridaCompleta.java. Integre tudo e calcule a classificação final usando operadores bit a bit para flags de penalidade.

1. Flags de penalidade (use operadores bit a bit):
final int PEN_VELOCIDADE = 1; // 0001 — excesso de velocidade final int PEN_COLISAO = 2; // 0010 — colisão final int PEN_PITLANE = 4; // 0100 — speeding no pitlane final int PEN_BANDEIRA = 8; // 1000 — ignorou bandeira int penalidades = PEN_VELOCIDADE | PEN_PITLANE; // piloto levou 2 penalidades
2. Cálculo de pontuação: posição final determina pontos (use ternário aninhado)
  → 1°=25, 2°=18, 3°=15, 4°=12, 5°=10, 6°+=0
  → Se tiver PEN_COLISAO: pontos -= 5; se PEN_BANDEIRA: pontos -= 10

3. Relatório final com posição, tempo total, número de pit stops, penalidades ativas e pontos marcados

Saída esperada (trecho):
