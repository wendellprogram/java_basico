Atenção

Divisão Inteira vs. Real

Em Java, dividir dois inteiros sempre resulta em um inteiro — a parte decimal é descartada. Para obter o resultado decimal, pelo menos um operando deve ser double.

JAVA 
int a = 7, b = 2;

// Divisão inteira — resultado: 3 (descarta o .5)
int inteiro = a / b;
System.out.println(inteiro);   // 3

// Divisão real — cast para double
double real = (double) a / b;
System.out.println(real);      // 3.5

// Operador módulo — resto da divisão
int resto = a % b;
System.out.println(resto);     // 1
💡
Dica prática: O operador % (módulo) é muito útil para saber se um número é par ou ímpar: numero % 2 == 0 retorna true se for par.
Código Completo

Exemplo com Todos os Operadores

JAVA — Operadores.java 
public class Operadores {
    public static void main(String[] args) {
        int x = 10, y = 3;

        // Aritméticos
        System.out.println(x + y);   // 13
        System.out.println(x - y);   // 7
        System.out.println(x * y);   // 30
        System.out.println(x / y);   // 3
        System.out.println(x % y);   // 1

        // Relacionais
        System.out.println(x > y);    // true
        System.out.println(x == y);   // false

        // Incremento/Decremento
        x++;
        System.out.println(x);        // 11

        // Atribuição composta
        y += 5;
        System.out.println(y);        // 8
    }
}
